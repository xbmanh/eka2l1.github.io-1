    Đối với Windows

    Phiên bản Windows yêu cầu bạn đã cài đặt phiên bản mới nhất của Microsoft Visual C++. Nếu khi bạn khởi chạy trình giả lập mà xuất hiện thông báo thiếu DLL, hãy thử tải xuống cài đặt runtime trên tại đây

    Đối với Android

    Hiện nay, hỗ trợ với Android (đặc biệt là 32-bit) còn đang trong giai đoạn thử nghiệm, do vậy, số lượng ứng dụng có thể hoạt động bình thường sẽ có thể khác xa phiên bản máy tính.

    Khi bạn gặp trường hợp ứng dụng có thể chạy trên máy tính nhưng không thể chạy trên Android, hãy liên hệ với nhà phát triển qua Discord, hoặc mở một issue trên Github.
